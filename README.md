# SC_BackupTool
Star Citizen user profile, controls and character backup script. batch file to backup control keybinds and user settings to user documents directory transfer between Live and PTU
##1. Place the batch file into users documents folder.
##2. Create a shortcut to the BAT file to place on the desktop
##3. configure shortcut to run as administrator or with admin access.
It is requrired to run as admin to ensure that the program files directories can be accessed by the script.
#IMPORTANT - please read the code and ensure that you are happy running before granting administrative privlidges.
